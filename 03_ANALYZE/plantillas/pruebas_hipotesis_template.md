## Guía de Selección de Prueba

| Situación | Tipo de datos | Prueba recomendada |
|-----------|--------------|-------------------|
| Comparar una media con valor objetivo | Continuos | t-test de 1 muestra |
| Comparar 2 medias (grupos independientes) | Continuos | t-test de 2 muestras |
| Comparar 2 medias (datos pareados) | Continuos | t-test pareado |
| Comparar ≥3 medias | Continuos | ANOVA |
| Relación entre 2 variables | Continuos | Correlación / Regresión |
| Comparar proporciones | Discretos | Prueba Z de proporciones |
| Asociación entre categorías | Discretos | Chi-cuadrado |
| Datos no normales | Cualquiera | Prueba no paramétrica equivalente |
